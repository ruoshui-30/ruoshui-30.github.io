1. 为什么前端引入图片之后会变模糊了


前端引入图片后变模糊，可能有以下几个原因：

  • 图片压缩：在上传图片时，为了减小图片文件大小，通常会进行图片压

缩，这会导致图片失真、变模糊。可以通过选择较高的压缩质量或者避免

过度压缩来减少失真。

  • 图片尺寸不匹配：如果网页中的图片大小与实际图片尺寸不匹配，浏览

器会对图片进行拉伸或缩放，从而导致图片失真、模糊。建议在使用图片

前，先确认图片大小是否与网页布局匹配。

  • 屏幕分辨率不同：如果使用的设备分辨率与图片分辨率不同，也会导致

图片模糊。这时可以使用更高分辨率的图片来解决问题。

  • 图片格式不当：不同的图片格式适用于不同的场景。比如，PNG 格式适

合处理透明背景的图片，而 JPEG 格式则适合处理色彩丰富的照片。选择

适合场景的图片格式可以提高图片的清晰度。

综上，前端引入图片模糊可能是由多种原因造成的，需要根据具体情况进行调整和优化。

优化

如果前端引入的图片出现了模糊的情况，可以考虑以下几个优化方法：

  1. 更换高质量的图片：如果图片失真、变模糊是由于压缩过度等原因造成

的，可以尝试更换较高质量的图片。使用高清晰度的图片可以提高图片的

清晰度，但需要注意图片大小的影响。

  1. 调整图片尺寸：在网页中展示的图片尺寸通常要比原始图片的尺寸小，

如果图片尺寸不匹配导致了图片模糊，可以根据实际需要调整图片尺寸。

建议在调整图片尺寸时，保持长宽比不变，避免拉伸变形。

  1. 选择合适的图片格式：不同的图片格式适用于不同的场景。选择合适的

图片格式可以提高图片的清晰度。比如，PNG 格式适合处理透明背景的图

片，而 JPEG 格式则适合处理色彩丰富的照片。

  1. 使用 CSS 技术：可以使用 CSS 技术来优化图片的显示效果。比如，

可以使用image-rendering属性来控制图片的渲染方式，以达到更好的清

晰度。还可以使用-webkit-font-smoothing属性来提高字体的清晰度。

  1. 使用图片优化工具：可以使用各种图片优化工具，例如

ImageMagick、Photoshop、TinyPNG 等，来对图片进行优化，以提高图

片的清晰度和质量。

总之，要优化图片的清晰度，需要结合具体情况，采取不同的优化方法。

最终目的是尽可能提高图片的清晰度，同时保持图片大小合理，以提高用

户体验。

2. 图片懒加载技术

图片懒加载技术是一种优化网页加载速度的技术，可以有效减少网页加载

时间，提高用户体验。图片懒加载技术的原理是，只有当用户滚动到图片

所在的位置时，才会加载图片。这样可以避免在网页加载时，一次性加载

大量图片，从而提高网页加载速度。

优点

jQuery懒加载技术可以提供以下几个优点：

  1. 提高页面加载速度：当页面中有大量图片或其他资源需要加载时，使用懒加载可以避免一次性加载所有资源导致页面加载速度过慢的问题。它可以在用户滚动到某个位置或触发某个事件时才加载对应的资源，从而提高页面加载速度和用户体验。
  2. 减少服务器负担：懒加载技术可以减少服务器的负担，因为服务器只需要在用户需要时才提供相应的资源，而不是一次性提供所有资源。这有助于降低服务器的带宽和流量成本。
  3. 节约带宽和流量：当用户只查看页面的一部分时，使用懒加载可以避免不必要的资源加载，从而节约带宽和流量。提高用户体验：使用懒加载技术可以让用户更快地看到页面内容，从而提高用户体验和满意度。
    总之，使用懒加载技术可以提高页面加载速度，减少服务器负担，节约带宽和流量，提高用户体验。

Intersection Observer API

Intersection Observer API 是一种新的 JavaScript API，用于监听目标

元素与其祖先元素或顶级文档视窗（viewport）交叉状态的变化。这

个 API 可以用来判断一个目标元素是否出现在视窗中，从而实现图片懒

加载的功能。
