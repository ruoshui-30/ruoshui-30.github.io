五、windows激活工具

盘点8款开源的Windows激活工具，代码是完全开源的。

  • MAS（推荐）：全球第一的Windows、Office激活工具，俄罗斯的专业团队开发的，并在今年推出了TSforge激活方式，有中文版本。

  • Ohook（推荐）：专业激活Office的一种方式，微软激活脚本MAS已提供这种方法。（离线激活Office的方法，不是激活Windows的，了解即可）

  • HEU KMS Activator （推荐）：国产第一的Windows、Office激活工具，默认是数字专业激活，使用人数在国内还是蛮多的。

  • 云萌数字权利激活工具：专业的数字激活工具，只能用于激活Windows，最大的特点是切换版本非常方便，良心出品，代码完全开源。

  • Activation Program：专业的离线激活Windows软件，仅支持TSForge。俄罗斯大神Ratiborus的一款激活软件，从软件来看，整个程序是基于MASSGRAVE的代码。

  • KMS服务器部署：需要自己部署KMS服务器，推荐这2个开源的KMS服务器搭建工具Py-KMS、Docker-KMS。有兴趣可以自己动手试试，非专业人员不用考虑这个了

1.MAS（微软激活脚本）

Microsoft Activation Scripts是完全开源的。

开源地址：

https://github.com/massgravel/Microsoft-Activation-Scripts

官网地址：

https://massgrave.dev

目前Github标星13.9万，在Github长期以来都是最热门的开源项目之一。

2.云萌数字权利激活

目前Windows10、Windows11使用最多的激活方式就是数字权利激活。

这款工具是专门针对数字激活的， 另外这款工具最好用的功能是可以一键切换所有Windows版本。

开源地址：

https://github.com/TGSAN/CMWTAT_Digital_Edition【https://github.com/TGSAN/CMWTAT_Digital_Edition】

官方网站：

云萌 Windows 10+ 激活工具 - 首页【https://cmwtat.cloudmoe.com/cn.html】

3.HEU KMS Activator

大名鼎鼎的HEU KMS Activator,虽然名字带有KMS,实际它默认使用的激活方式是数字激活，也是Win10、Win11使用的激活方式最多的方式。

国产同类第一的激活软件，也是国内用户使用人数最多的工具——HEU KMS Activator

使用HEU可以选择的离线激活Windows方法有【TForge】、【KMS38】。

下载链接：HEU_KMS_Activator【https://github.com/zbezj/HEU_KMS_Activator/releases】

4.ohook

一种专业离线激活Office方法，微软激活脚本MAS、HEU KMS Activator已经集成了这种方法。

有兴趣的可以下载了解一下，开源地址：

ohook【https://github.com/asdcorp/ohook】

OHook激活特点：

一次使用，永久激活，无需像KMS 那样定期激活（不过KMS38也是无需联网激活的）；

支持Office 2013-2021、Office 365。office2010及以下不可用（2007及以下都是序列号激活，所以只有2010这个独苗了）；

解锁Office全部功能（除联网功能）；

离线激活、无需外网，符合你的要求；

不支持Windows 7及以下系统；

5.Activation Program

俄罗斯大神Ratiborus的一款激活软件，从软件来看，整个程序是基于MASSGRAVE的代码，激活方法采用的TSForge。

激活方法参考网址：这8个Windows激活的开源项目，霸榜Github！ - 知乎【https://zhuanlan.zhihu.com/p/1943476358459166787】

6.在线KMS激活命令

不开源，但是可以直接复制KMS激活命令

关于KMS激活必须要了解这3点：

1、KMS激活是需要联网的，激活时间只有180天；

2、除了联网KMS激活，还是KMS38激活，无需联网，属于是永久激活的；

3、目前Windows10、Windows11主流的激活方式不是KMS激活，是数字权利激活。

KMS激活必备的2个条件：

1、必须以管理员方式运行命令行提示符；

2、需要公开的KMS服务器地址+Windows的激活密钥（有些可以不用输入）

网站地址：一键激活windows系统和office软件,Office软件安装,KMS激活【https://kms.cx/】

7.开源的KMS服务器搭建

KMS激活是微软为企业、组织提供的批量激活方式，企业、组织通过像微软购买许可，架设 KMS 激活服务器就可供其成员激活 Windows 和 Office 。

KMS全称：Key Management Service。

通过KMS，组织可以在内部搭建激活服务器，管理大量计算机的激活状态。

搭建KMS激活服务器是需要取得微软的官方授权的，一般个人是肯定无法获取授权，至少网上那些公开的KMS激活服务器，一般都是来自某些组织或学校。

如果你有需求搭建KMS服务器，可以参考这个开源项目——Py-KMS。

开源地址：py-KMS【https://github.com/Py-KMS-Organization/py-kms】

8.Docker-KMS

Docker版本的KMS服务器，也是给予Py-KMS的。

开源地址：

Docker-KMS【https://github.com/11notes/docker-kms】

在使用 KMS 主机的环境中，批量激活过程如下：

  1. 将 KMS 安装到服务器上
  2. 将 KMS 主机密钥安装到 KMS 主机，然后联系微软进行激活
  3. KMS 启用后，每天启动的时候会在 DNS 中主次一条 SRV 记录
  4. 客户端通过配置的注册条目或者 DNS 中的 KMS SRV 记录发现 KMS 主机
  5. 客户端向 KMS 服务器的 TCP/1688 发送 RPC 请求，请求中包含一个加密的计算机 ID。如果该计算机尚未激活，并且 KMS 主机没有响应，客户端 2 小时后重新发送 请求。如果计算机已激活，客户端在 7 天后发送一个新请求。
  6. KMS 服务器将客户端计算机 ID 添加到一个表格中，并将激活计数返回给客户端
  7. 客户端将激活计数值与授权策略对比，如果满足激活阀值条件，则进行激活。

通过以上步骤就可以激活你的Windows系统和Office软件（2小时内），但是值得注意的是，KMS方式激活的有效期只有180天，因此每不到半年需要重新激活一次。
